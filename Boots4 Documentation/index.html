Getting Started

Unzip the downloaded folder and find out the assets. There are all the template files in this folder. You can open the following files for viewing in browser.

  1. css — folder with css files.
  2. img — folder with image files and png Icon.
  3. js — folder with Javascript files.
  4. index.html — the only html file contains all feature of this single page awesome template.

Home Screen Section

There are four types of home page based on logo and navigation-item number. One is text logo and another is regular image logo. As a result, you will find 4 html pages.

The width & height of home page is decided by javascript. In script.js file you'll find the code from line 24 to 41 

    var windowsWidth = $(window).width(),
        windowsHeight = $(window).height();

    if (windowsWidth > 767) {

        $('.introduction , .menu').css({
            width: '50%',
            height: '100%'
        });

    } else {

        $('.introduction , .menu').css({
            width: '100%',
            height: '50%'
        });

    }

Again menu item number can also be changed. For our customer's convenience, we provide two types of html pages. If you want to change menu item, please folow the following steps:

  • Add or delete nav-item like following line of code:
    •     <div data-url_target="YOUR_NAV_ITEM_TARGET" class="NAV_NAME_BTN menu_button">
        <img alt="" src="YOUR_IMAGE_PATH">
        <div class="mask">
        </div>
        <div class="heading">
            <i class="NAV_ITEM_ICON hidden-xs"></i>
            <h2>NAV_ITEM_TITLE</h2>
        </div>
    </div>
  • Add or delete content of that corresponding nav-item like other sections or pages whatever you say. The structure is like following:
    •     <div id="YOUR_NAV_ITEM_TARGET" class="NAV_TITLE_CLASS container-fluid page">
        <div class="row">
            <div class="image-container col-md-3 col-sm-12">
                <div class="mask"></div>
                <div class="main-heading"><h1>Your Page Title</h1></div>
            </div>
            <div class="content-container">
                <p>.......... (your content)</p>
            </div>
        </div>
    </div>
  • Now, change the css from style.css according to your content. Like for 4 nav-items, we used a class called four_nav_item and for six nav items, this class name is changed to six_nav_item. And the css is used like following in line from 588 to 594
    •     .four_nav_item.menu > div {
        height: 50%;
    }

    .six_nav_item.menu > div {
        height: 33.33%;
    }
  • Again, if you use new class name, you have to change the css as your wish

Close Button

In index.html file. This code placed just under the end of Home Screen Section. This button shows on every section to close this section and back to theHomescreen Section 

    <div class="close-btn"></div>

The functionality is controlled from script js. You will find the code from line 196 to 209

    // Close Button, Hide Menu

    $('body').on('click', '.close-btn', function () {
        window.location.hash="";
        $('.home-page').css({
            display: 'block'
        });
        $('.introduction, .menu').animate({
            left: 0
        }, 1000, 'easeOutQuart');
        $('.page').fadeOut(800);
        removeHash ();
        $(window).scrollTop(0);
    });

Counter in About Us Page

You'll find counter in fun fact section of about us page. Counter will start after loading the about us page. It is also a result of some part of javascript code of script.js The code, responsible for this section, will be found from line 121 to 135

    $('.menu').on('click', 'div.profile-btn', function () {
        setTimeout(function(){
            $('.count').each(function () {
                $(this).prop('Counter',0).animate({
                    Counter: $(this).text()
                }, {
                    duration: 1500,
                    easing: 'swing',
                    step: function (now) {
                        $(this).text(Math.ceil(now));
                    }
                });
            });
        }, 100);
    });

If Contact form form not working

You need to check is PHP mail() function working.

  • Save this code as mailtest.php
  • change you@yourmail.com to your e-mail address
  • upload mailtest.php to your server
  • open mailtest.php in your browser (http://yourwebsite.com/mailtest.php)
  • check your inbox to see if a test message arrived.
If it works:
  • double-check your form script for errors (like e-mail address misspelling)
  • use the same e-mail address as your form recipient
  • double-check your SPAM filters and SPAM/Junk/Bulk mailboxes
If it not

Contact your host and ask them to check PHP mail() setting.

Google Map

Recently Google has changed the system. One must have an API key for a domain to show google map in any website of that domain. So you have to create an API key for your website to take this service of google.

In our template, you will find the script for google map, added in the bottom of the file like following:

    <script src="https://maps.googleapis.com/maps/api/js"></script>

Please change that line like this

    <script async defer src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap"></script>

For more to know, following links will help you more to get the API key for google map for your site.

  • Steps to get an API key
  • Steps to add a marker in map

If you want to change the map location, just change the latitude and longitude of the script file added at the last in html pages.

    <script>
      function initMap() {
        var location = {lat: YOUR_LATITUDE, lng: YOUR_LONGITUDE};
        var map = new google.maps.Map(document.getElementById('map-canvas'), {
            zoom: 14,
            center: location,
            scrollwheel: false,
            mapTypeId: google.maps.MapTypeId.ROADMAP
        });
        var marker = new google.maps.Marker({
            position: location,
            map: map,
            title:"Boots4 Office"
        });
    }
    </script>
